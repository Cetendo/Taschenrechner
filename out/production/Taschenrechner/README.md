## Curent Features
* Keystroks 
  * `ESCAPE` for C
  * `DELETE` for CE
  * `BACK_SPACE` for DEL
  * `NUMPAD0` to `NUMPAD9` for 0 - 9
  * `DECIMAL` (Numpad) for ,
  * `ADD` (Numpad) for +
  * `SUBTRACT` (Numpad) for -
  * `MULTIPLY` (Numpad) for *
  * `DIVIDE` (Numpad) for /
  * `ENTER` (Numpad and Normal) for =
* Window Resizing
* Calculation
